ngx-slides

Custom Dropdown component for Angular 2+ applications with multiple and single selection options.

1.Installation

        //using npm
         npm install ngx-slides

        //using yarn
         yarn add ngx-slides
         yarn install
  • include styles in you angular-cli.json.
        //in angular-cli.json
         "styles": [
            "../node_modules/ngx-slides/dist/assets/style.css"
          ]

2.Using with webpack and tsc builds/ angular-cli builds

  • import NgxSlidesModule from ngx-slides
  • add NgxSlidesModule to the imports of your NgModule
        
          import {{'{'}} NgxSlidesModule {{'}'}} from 'ngx-slides'
        
          @NgModule({{'{'}}
            imports: [
              ...,
          
              NgxSlidesModule
            ],
            ...
          })
          class YourModule {{'{'}} ... {{'}'}}
  • use <ngx-slides></ngx-slides> in your templates to add the slider in your view like below
  • a tag in you html will look like below with all the config
          <ngx-slides [config]="config" [slideImages]="slideImages" ></ngx-slides>

3.Config

Input
  • slideImages: any[] - array of uri's for the images in the slides 
        config = {{'[{'}}
          image: '/assets/image2.jpg', //image uri
          title: "Image 2" //image title
          {{'}]'}}
  • config: Object - configuration object. 
          config = {{'{'}}
            width:"800px" //width of the slider defaults to 800px
            height:"300px" //height of the slider defaults to 300px
            autoPlay: true/false // base on whether autoplay required or not
            delay:3000 //delay for autoplay in ms defaults to 3000ms
            {{'}'}}
